# Std-Research
聚焦平顶山标准化博士团队的课题研究申报相关工作，汇聚尹国友、孙婕、岳永强等核心专家的研究成果与课题资源。团队成员涵盖国际标准化组织中方专家、海外访问学者、工商管理博士等多元背景，研究领域覆盖标准化体系构建、食品加工与功能性食品研发、绿色城配领域 GS1 系统应用等方向。仓库收录了团队主持 / 参与的 20 余项科研项目、多部专著与专利成果，以及教育部产学合作协同育人项目、省级科技奖项等核心资料，同时开放课题研究参与申请通道（私信 “践行标准化” 公众号，邮箱674348396@qq.com）。旨在整合标准化研究资源，推动产学研协同创新，为平顶山乃至全国标准化领域的研究与实践提供参考和协作平台。
